# End-to-End Customer Churn Prediction System

## Overview
Built a machine learning pipeline to predict customer churn using XGBoost and Optuna.

## Dataset
- 7,043 customers
- 21 features
- 26.55% churn rate

## Results
- F1 Score: 0.58
- Recall (Churn Class): 0.75

## Tech Stack
- Python
- Scikit-learn
- XGBoost
- Optuna
- MLflow
- SHAP

## How to Run
pip install -r requirements.txt
jupyter notebook
